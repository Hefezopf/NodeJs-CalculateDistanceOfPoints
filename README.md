# NodeJsCalcPoints
The Javascript NodeJs Program calculates the minimum distance of points.

## Prepare
npm install

## Run
### CMD Version
node src/nodeJsCalcPointsApp.js

### Web Version
node src/nodeJsCalcPointsWeb (or with extension .js)

node src/nodeJsCalcPointsWeb.js

http://localhost:8080/

## Test
npm test
